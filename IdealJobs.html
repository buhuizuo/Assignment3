IDEAL JOBS

SOFTWARE DEVELOPERS

builds software which runs across different types of computer. They write code from scratch. The application could be a desktop application mobile apps, web apps.

WEB DEVELOPERS

coding for a website or a web application. A website consists of 3 tiers. Based on team size and development model.

AUTOMATION ENGINEER

delivers automated solutions for software processes.

Job detail

WEB DEVELOPERS

• designing and developing effective software&applications.
• analysis, maintenance and remediation of existing systems.
• support and troubleshooting.

SOFTWARE DEVELOPERS

• analysis, design, development, support, and enhancement of software products services.
• building web applications.
• Front-end and back-end development activities.
• building systems fromscratch and enhancing existing systems .
• production support and troubleshooting.

AUTOMATION ENGINEER

• development of machines that automatically control appliances.
• designing simple diagnostics.
